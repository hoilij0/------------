Что такое HTML?

HTML (HyperText Markup Language) — это стандартный язык разметки, который используется для создания веб-страниц. Он определяет структуру документа, а внешний вид задаётся с помощью CSS.

История языка

HTML был создан Тимом Бернерсом-Ли в 1991 году. Первая версия позволяла создавать простые страницы с заголовками, параграфами и ссылками. Современные версии, начиная с HTML5, поддерживают видео, аудио, семантические теги и многое другое.

Пример HTML-кода

<!DOCTYPE html>
<html>
<head>
    <title>Пример страницы</title>
</head>
<body>
    <h1>Здравствуйте!</h1>
    <p>Это пример простой HTML-страницы.</p>
</body>
</html>
